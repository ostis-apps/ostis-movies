Покебол

Покебол — предмет, который имеет решающее значение для тренера, используется для ловли и хранения покемонов. Как общий термин, используемый для описания различных видов, так и конкретный термин для обозначения наиболее основных из этих вариаций, покеболы повсеместно распространены в современном мире покемонов.
