Внешность

В отличие от своих предэволюций, у Драгонайта есть конечности. Всем своим видом он напоминает драконов из мифов Средневековья. Несмотря на то, что он уменьшается после эволюции, Драгонайт всё равно больше человека. С помощью маленьких крыльев и большого тела Драгонайт может летать быстрее скорости звука.
