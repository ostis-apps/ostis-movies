Krallahi fight with rage like wolves or feral dogs. The Axii sign can weaken them, but only for a moment. Hounds hunt in packs and can quickly crush untrained foes. The witcher should not allow them to surround him, as well as, if possible, use the Yrden sign to slow down hounds that are too close, and maintain a distance, if necessary.


The icy nature of hounds makes them ideal targets for the sign of Igni. It also makes them feel energized by any cold environment. You should step back from the hounds if they start to form ice spikes. These obstacles will break through the attacker if you try to strike the hound.
