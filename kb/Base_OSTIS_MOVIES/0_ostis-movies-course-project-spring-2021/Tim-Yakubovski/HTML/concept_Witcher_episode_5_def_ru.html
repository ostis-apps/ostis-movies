Геральт с помощью сети разыскивает кувшин с джинном. Лютик случайно разбивает кувшин и заражается магической болезнью. Для исцеления своего спутника Геральт прибегает к помощи чародейки Йеннифэр из Венгерберга. Кагыр, чтобы найти Цириллу, прибегает к услугам допплера, который принимает облик Мышовура.
