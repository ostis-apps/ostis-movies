Полное имя героини — Цирилла Фиона Элен Рианнон. Носит также прозвище Львенок из Цинтры. Имя героини на Старшей Речи звучит как «Зираэль», что означает «ласточка». В саге о Ведьмаке установлено собственное летоисчисление, согласно которому Цири родилась в мае 1252 года.Цири становится объектом интереса множества влиятельных людей, каждый из которых имеет на девочку собственные планы. На героиню пытаются «наложить руку» нильфгаардский император, Ложа чародеек и колдун Вильгефорц. Причина такого интереса — предсказание, согласно которому потомок Цири повлияет на судьбы мира.
