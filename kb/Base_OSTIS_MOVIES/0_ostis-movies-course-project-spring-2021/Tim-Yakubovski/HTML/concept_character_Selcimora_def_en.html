According to rumors, the mouth of this creature is dotted with thousands of sharp teeth, and this creature is able to swallow everything that comes its way. There is no reliable data on what exactly this monster looks like, but there are stories in which entire villages disappeared into its bottomless mouth. According to legend, a more detailed description of Selkimora can be given by the witcher Geralt, who, before killing her, examined this beast both from the outside and from the inside.
