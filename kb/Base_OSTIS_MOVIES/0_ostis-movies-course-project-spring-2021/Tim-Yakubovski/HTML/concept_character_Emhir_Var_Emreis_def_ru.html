Император Эмгыр вар Эмрейс

Император Нильфгаарда или просто Император (на нифльгаардском диалекте — Ker'zaer) — титул правителя Нильфгаардской Империи, передаваемый по наследству. Носитель этого титула обладает неограниченной властью. Политическую власть он соединяет с религиозной — император является верховным жрецом Великого Солнца, то есть главой доминирующего в Империи культа. Сенат Нильфгаарда не оказывает на него никакого влияния. За личную безопасность императора отвечает бригада Импера, насчитывающая около 5000 фанатично преданных государю солдат.
