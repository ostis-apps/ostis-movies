Лютик , он же Юлиан Альфред Панкрац, виконт де Леттенхоф — странствующий бард. Лучший друг и неизменный спутник Геральта из Ривии. Лютик — автор многочисленных баллад и стихов. Во многих своих произведениях он повествует о приключениях Геральта, благодаря чему ведьмак довольно знаменит, фактически являясь самым известным ведьмаком Севера. Более-менее полно читателям становятся известны лишь отрывки из мемуаров «Полвека поэзии», которые Лютик начал писать после дезертирства Геральта из ривской армии. Очевидно, эти мемуары — один из немногих переживших ледниковый период памятников литературы северян.
