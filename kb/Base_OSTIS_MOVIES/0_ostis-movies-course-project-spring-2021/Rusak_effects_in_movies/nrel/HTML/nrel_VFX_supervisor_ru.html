VFx-супервайзер — это такой специалист, который понимает, какого эффекта в кадре нужно добиться, может придумать (или уже знает), что конкретно нужно сделать, чтобы получить результат, может разработать план действий для других людей, и проконтролировать процесс создания результата от начала и до конца, решая возникающие по ходу дела нюансы.
