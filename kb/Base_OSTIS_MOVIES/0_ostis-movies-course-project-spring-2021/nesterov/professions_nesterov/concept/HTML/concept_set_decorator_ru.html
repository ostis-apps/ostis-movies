Художник по декорациям, часто являясь художником с архитектурным образованием, вычерчивает планы и составляет спецификации по материалам для строительства декораций на основе словесного описания или эскизов, предоставляемых художественным руководителем. Из-за высокой стоимости работ по строительству декораций художник по декорациям при планировании старается предусмотреть строительство только тех объектов или их элементов, которые реально попадут в объектив кинокамеры.
