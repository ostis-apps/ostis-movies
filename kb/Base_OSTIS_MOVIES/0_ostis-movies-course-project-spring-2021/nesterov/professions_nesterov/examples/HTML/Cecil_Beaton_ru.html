Сэр Сесил Уолтер Харди Битон, более известный как Сесил Битон — известный английский фотограф, мемуарист, икона стиля, дизайнер интерьеров, художник по костюмам и декорациям, один из важнейших мастеров модной фотографии. В 1970 году был включён в Международный список Зала славы самых стильных людей.
