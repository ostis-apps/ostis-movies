Красное дерево, также известное как просто Мясник, является злодейской личностью и главным антагонистом рассказа «Полуночный мясной поезд» из первого тома «Книги крови», позже адаптированного для одноименного фильма ужасов 2008 года, где он наиболее известен. Он безжалостный серийный убийца, у которого есть особенность совершения своих ужасных действий на линиях метро, всегда вскрывая тела своих жертв и подвешивая их к ногам на ручках. Однако его убийства - необходимое зло, чтобы предотвратить атаку более серьезной угрозы, которая живет под поверхностью.
