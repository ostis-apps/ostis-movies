Леон Кински когда-то был достаточно нормальным человеком, который, движимый романтическими чувствами к недостижимому любовнику, был вынужден выпить эликсир, чтобы изменить свою внешность, чтобы больше нравиться объекту его привязанностей. Вместо этого, уже испытав воздействие сквозняка, он был очарован полумесяцем луны с непредсказуемым и очевидным результатом.
