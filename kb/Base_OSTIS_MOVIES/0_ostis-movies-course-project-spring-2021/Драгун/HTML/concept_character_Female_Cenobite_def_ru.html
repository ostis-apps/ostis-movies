Незнакомец, нашедший убежище в её церкви, ощутил её потребности и дал Николетте дар Головоломки (англ. Puzzle Box). Отличительные особенности: безволосая иссиня-бледная голова, нос проткнут гвоздём, из разреза в горле отходит проволочная конструкция, продетая сквозь щёки.
