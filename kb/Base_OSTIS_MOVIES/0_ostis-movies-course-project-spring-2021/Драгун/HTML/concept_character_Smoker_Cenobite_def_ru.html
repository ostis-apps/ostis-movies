женщина-псевдосенобит Терри, бездомная подружка Джей Пи Монро, нашедшая для него скульптуру Pillar Of Soul (рус. Столб Душ) в галерее «Пирамида». Освободила Пинхэда из скульптуры, отдав ему на растерзание своего бойфренда. Отличительные особенности: её собственное лицо натянуто на голову, как маска; из разреза в её шее торчит дымящаяся сигарета, а с кистей полностью содрана кожа.
